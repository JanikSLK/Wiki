# ฟีเจอร์ \(Features\)

## เกี่ยวกับ

**PreMiD** นี่ง่ายมากเลยนะ เป็นโปรแกรมที่แสดงว่า คุณ "ดู"หรือ"ฟัง"อะไรอยู่ ใน Discord  **\(Now Playing..\)** โปรแกรมของเรารองรับ[หลายแพลตฟอร์ม](support/services.md) และรองรับการดูวิดีโอพร้อมกันกับสหายของคุณในอัปเดตที่กำลังจะมาถึง

## ฟีเจอร์

* ปรับความเข้ากันได้กับ API ของ Discord Library \(Integrating\)
* รองรับการ หยุด/เล่น ผ่านปุ่มคีย์บอร์ดของคุณ \(Media Control\)
* เมื่อคุณไม่ได้ใช้งานอยู่ ระบบจะล้าง Presence ของคุณใน 1 นาที! \(Inactivity\)
* รองรับ **YouTube, Twitch, Netflix, SoundCloud** และอีกมากมายจะมาในเร็ว ๆ นี้!
* _ระบบปาร์ตี้ดูพร้อมกันจะมาในเร็ว ๆ นี้_
